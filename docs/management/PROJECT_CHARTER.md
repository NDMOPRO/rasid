# Project Charter

## اسم المشروع
Rasid National Platform

## الغرض
تطوير منصة تشغيلية وتحليلية موحدة لدعم الرقابة واتخاذ القرار.

## النطاق المختصر
- بناء واجهة مستخدم موحدة.
- بناء واجهة API آمنة.
- إدارة بيانات تشغيلية وتحليلية.

## أصحاب القرار
- Sponsor: الإدارة التنفيذية.
- Product Owner: إدارة التحول الرقمي.
- Engineering Lead: فريق التقنية.

## مؤشرات النجاح
- جاهزية تشغيلية كاملة.
- خفض زمن إنشاء التقارير.
- اعتماد تشغيلي من الفرق المستهدفة.
