# FRD - Functional Requirements Document

## الوحدات الوظيفية

## 1. إدارة الهوية والوصول
- FR-01: يجب أن يدعم النظام تسجيل الدخول بالبريد/اسم المستخدم + كلمة المرور.
- FR-02: يجب أن يطبق النظام صلاحيات بحسب الدور.
- FR-03: يجب تسجيل عمليات الدخول الفاشلة والناجحة.

## 2. إدارة المسوحات
- FR-10: إنشاء مسح جديد وتحديد الحالة والجهة.
- FR-11: إرفاق الأدلة والروابط والملفات.
- FR-12: تحديث حالة المسح (جديد/قيد التنفيذ/مكتمل/مؤرشف).

## 3. التحليلات والتقارير
- FR-20: عرض KPI على لوحة مؤشرات.
- FR-21: تصفية النتائج حسب الزمن والمنطقة والجهة.
- FR-22: تصدير التقرير PDF/Excel.

## 4. المساعد الذكي
- FR-30: دعم جلسات محادثة.
- FR-31: حفظ الرسائل وسجل التقييم.
- FR-32: دعم أوامر مخصصة.
