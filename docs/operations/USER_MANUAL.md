# User Manual

## 1. تسجيل الدخول
1. افتح رابط المنصة.
2. أدخل اسم المستخدم وكلمة المرور.
3. اضغط "دخول".

## 2. استخدام لوحة المؤشرات
- اختر الفلاتر الزمنية أو المناطق.
- راقب بطاقات KPI والرسوم البيانية.

## 3. إدارة المسوحات
- أنشئ مسحًا جديدًا من زر الإضافة.
- أضف الأدلة واربطها بالمسح.
- حدّث الحالة حتى الإغلاق.

## 4. التصدير
- انتقل إلى شاشة التقرير.
- اختر PDF أو Excel.
- نزّل الملف وشاركه مع المعنيين.

## 5. المساعد الذكي
- افتح نافذة AI.
- أرسل استفسارًا.
- راجع الإجابة واحفظ الجلسة.
